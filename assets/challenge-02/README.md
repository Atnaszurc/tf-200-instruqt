# Challenge 2: Advanced Module Patterns

This workspace demonstrates enterprise-grade module patterns.

## Modules

### Nested Modules
- **network**: Creates Libvirt networks
- **storage**: Creates storage pools and volumes
- **compute**: Creates VMs
- **app-stack**: Composes all three layers

### Advanced Patterns
- **conditional-vm**: Conditional resource creation
- **canary-deployment**: Gradual rollout pattern
