# Challenge 1: Module Design & Composition

This workspace contains example modules for learning Terraform module design.

## Modules

- **app-config**: Simple configuration file module
- **libvirt-network**: Network module for Libvirt
- **libvirt-vm**: VM module for Libvirt

## Getting Started

Follow the instructions in the challenge assignment to:
1. Understand module structure
2. Create and test modules
3. Compose modules together
4. Refactor with `moved` blocks

## Example Files

- `main.tf.example`: Example root module (rename to main.tf to use)
- `outputs.tf.example`: Example outputs (rename to outputs.tf to use)

Happy learning! 🚀
